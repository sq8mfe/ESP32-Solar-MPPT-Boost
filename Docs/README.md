# ESP32 Solar MPPT Boost Converter  
![Schemat](Hardware/Schematics/solar_mppt.png)  

## 📌 Features  
- MPPT algorithm (Perturb & Observe)  
- WiFi monitoring  
- Input: 3× 380W solar panels  

## 🔧 How to build  
1. Solder components from [BOM.csv](Hardware/BOM.csv).  
2. Flash firmware using [PlatformIO](Firmware/PlatformIO).
